# MasalaMeter Backend

Backend services for the MasalaMeter AI food scanner.

MasalaMeter analyzes meals and delivers a “royal verdict” inspired by the character Sivagami — combining food recognition, nutrition analysis, and a playful scoring engine.

This repository will power the core intelligence behind the MasalaMeter app.

## Planned Capabilities

• Food recognition from images  
• Ingredient extraction and translation  
• Nutrition analysis  
• Sugar and junk food detection  
• AI-powered “Truth Score” calculation  
• Roast-style verdict generation  
• Food comparison engine  

## Architecture (planned)

Frontend: MasalaMeter mobile/web UI  
Backend: Python / FastAPI or Flask  
AI Vision: Food recognition model or API  
Nutrition Data: Public nutrition databases  
Scoring Engine: Custom logic for food health scoring  

## API Endpoints (planned)

POST /scan-food  
POST /analyze-food  
GET /nutrition-data  
POST /compare-foods  

## Project Status

Early prototype phase.

The frontend experience and UI are currently being prototyped while backend AI services are under design.

## Vision

MasalaMeter aims to become the most engaging and shareable AI food scanner — combining nutrition insight with humor, personality, and cultural flavor.

Serving Truth, with a Side of Judgement.
