How TruCite Works


  

    TruCite evaluates claims using a deterministic evidence pipeline — not language confidence.
  


  


    

      
1. Claim Isolation

      
Extracts factual assertions from AI-generated text.

    


    

      
2. Evidence Retrieval

      
Queries vetted sources (academic, regulatory, primary datasets).

    


    

      
3. Truth Scoring

      
Scores claims based on evidence strength, consistency, and provenance.
